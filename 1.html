HTML

Hypertext Maarkup Language (HTML) is the standard markup language for creating web pages and web application.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactice forms, may be embedded into the rendered page. It procides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links,quotes and other items. HTML elements are delineated by tags, written using angle brackets.
